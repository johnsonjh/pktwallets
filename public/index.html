  • PKT‑Electrum is a freely available, multi-platform, low‑resource, low‑bandwidth, high‑speed, lightweight, and secure wallet solution for the PKT Cash blockchain.

  • PKT‑Electrum was built to meet the needs of crypto‑currency novices as well as advanced users. It provides a graphical user interface as well as command‑line interface.

  • PKT‑Electrum supports advanced features such as multi‑signature transactions, support for off‑line cold storage wallets, and transaction obfuscation designed to enhance privacy.

  • PKT‑Electrum synchronizes fast, accelerated by the use of specialized servers. Because it doesn't need to download and store the full PKT Cash blockchain, most users can get setup in a matter of minutes.

  • PKT‑Electrum is only a wallet. It can not mine PKT Cash. Because it was not designed for high volume transaction processing, and should never be used to directly receive mining proceeds.
  • PKT‑Electrum releases are digitally signed using PGP/GnuPG, Signify/Minisign, oa8sign, and opmsg for your protection.
