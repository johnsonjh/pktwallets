  • PKT-Electrum is a multiplatform, freely available, fast, and lightweight wallet available that works with the PKT Cash blockchain.

  • PKT-Electrum is designed to suit the needs of both beginners and advanced users. It offers both GUI and command-line interfaces.

  • PKT-Electrum supports advanced features like multi-signature transactions, offline cold storage, and transaction obfuscation for enhanced user privacy.

  • PKT-Electrum synchronizes fast, does not require fully downloading and storing a full copy of the blockchain, and can be setup in a matter of minutes.
  • PKT-Electrum is only a wallet. It does not mine PKT Cash. It was not designed for high-volume transaction processing, and should not be used for directly receiving mining proceeds.

  • PKTWALLET is a multiplatform, freely available, advanced, programmable wallet available that works with the PKT Cash blockchain.

  • PKTWALLET was designed to meet the needs of advanced users and programmers. It offers a UNIX-style command-line interface, and connectivity with JSON gRPC, BTC-Core, and BTCD-PKTD full node RPC compatible interfaces. It has it's own lightweight P2P mode, which is based on the Neutrino protocol.

  • PKTWALLET supports every advanced feature of the PKT network, such as Network Steward voting, multi-signature transactions, cold storage, and advanced scripting, logging, and debugging facilities.

  • PKTWALLET directly interfaces with the PKTD full node server daemon, offering advanced users full control.

  • PKTWALLET is also only a wallet. It can not mine PKT. However, it was designed with the needs of those needing high volume transaction processing capabilities or integration with mining pools and exchanges. It is commonly used with the PacketCrypt mining software.
